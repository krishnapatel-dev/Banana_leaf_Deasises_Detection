# Banana Leaf Disease Detection

Deep learning project for classifying banana leaf diseases using a CNN model built with TensorFlow.

--------------------------------------------------

REQUIREMENTS

- Python 3.8+
- tensorflow
- albumentations
- opencv-python-headless
- numpy
- matplotlib
- scikit-learn
- tqdm

Install dependencies using:

pip install tensorflow albumentations opencv-python-headless numpy matplotlib scikit-learn tqdm

--------------------------------------------------

MODULES USED

- tensorflow        → Model building and training
- albumentations    → Image augmentation
- opencv (cv2)      → Image processing
- numpy             → Numerical operations
- matplotlib        → Visualization
- sklearn           → Data preprocessing and evaluation
- tqdm              → Progress bar

--------------------------------------------------

HOW TO RUN

1. Clone the repository:

git clone https://github.com/krishnapatel-dev/Banana_leaf_Deasises_Detection.git
cd Banana_leaf_Deasises_Detection

2. Place dataset in this structure:

Images/
├── Cordana/
├── Healthy/
├── Pestalotiopsis/
└── Sigatoka/

3. Open and run the notebook:

Banana_leaf_Deasises_Detection.ipynb

Run all cells sequentially.

Dataset : https://www.kaggle.com/datasets/shifatearman/bananalsd
